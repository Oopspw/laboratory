## Build Setup

```bash
# install dependencies
npm install 安装依赖包

# serve with hot reload at localhost:8080
npm run dev 编译开发环境

# build for production with minification
npm run build 编译生产环境

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit 启动单元测试

# run e2e tests
npm run e2e 启动端到端测试

# run all tests
npm test
```
